simgon.behavior_splitter

simgon.behavior_splitter.construct_features(data, significanceLevel=0.01)[source]

Construct feature vectors for time series data based on behavioral characteristics.

Parameters:

  • data (np.ndarray) – Input data as 2D array. Each row represents a time series.

  • significanceLevel (float, optional) – Threshold level for determining significant changes. Default is 0.01.

Returns:

A list where each element is a 2D array with shape (2, n_sections) containing slope and curvature signs for that series. The first row is slope sign, the second row is curvature sign. Lengths can vary by series.

Return type:

List[np.ndarray]

simgon.behavior_splitter.split_behavior(data_series, slope, curvature, significanceLevel=0.01)[source]

Split a time series into sections of different atomic behavior modes.

Parameters:

  • data_series (np.ndarray) – 1D array representing a single time series.

  • slope (np.ndarray) – 1D array containing the first derivative.

  • curvature (np.ndarray) – 1D array containing the second derivative.

  • significanceLevel (float, optional) – Threshold level for filtering out insignificant changes.

Returns:

2D array with shape (2, n_sections) containing slope and curvature signs. The first dimension (2) represents: [0] slope sign, [1] curvature sign.

Return type:

np.ndarray
