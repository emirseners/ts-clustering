API Reference

Welcome to the API Reference section. Here you’ll find detailed documentation for all modules and functions in the Time Series Clustering library.

Available Modules:

  • simgon.behavior_splitter - Behavioral feature extraction and splitting

  • simgon.clusterer - Main clustering functionality

  • simgon.distance_dtw - Dynamic Time Warping distance calculations

  • simgon.distance_pattern - Pattern-based distance metrics

  • simgon.distance_pattern_dtw - Pattern-based DTW distance metrics

Distance Metrics Reference

The following distance metrics are available through the distance='scipy' parameter in the simgon.clusterer.cluster() function:

Pattern-based distances: * simgon.distance_pattern.distance_pattern() - Pattern distance using behavioral features * simgon.distance_pattern_dtw.distance_pattern_dtw() - Pattern distance with Dynamic Time Warping * simgon.distance_dtw.distance_dtw() - Dynamic Time Warping distance

Scipy distance metrics: * simgon.distance_scipy.distance_scipy() - Access to all scipy.spatial.distance.pdist metrics

Note: The distance_scipy module is designed to be used internally through the simgon.clusterer.cluster() function. Users typically don’t need to import or use it directly. Instead, use distance=’scipy’ with additional parameters in the cluster function.

For detailed information about scipy distance metrics, see the scipy documentation.

Available Scipy Metrics:

The following metrics are directly available as string parameters in the cluster function:

  • ‘euclidean’ - Euclidean distance (L2 norm)

  • ‘manhattan’ / ‘cityblock’ - Manhattan distance (L1 norm)

  • ‘mse’ / ‘sse’ / ‘sqeuclidean’ - Squared Euclidean distance

  • ‘triangle’ / ‘cosine’ - Cosine distance

  • ‘correlation’ - Correlation distance

  • ‘chebyshev’ - Chebyshev distance (L∞ norm)

  • ‘canberra’ - Canberra distance

  • ‘braycurtis’ - Bray-Curtis distance

  • ‘hamming’ - Hamming distance

  • ‘jaccard’ - Jaccard distance

  • ‘minkowski’ - Minkowski distance (with p parameter)

  • ‘mahalanobis’ - Mahalanobis distance

  • ‘seuclidean’ - Standardized Euclidean distance

  • ‘dice’ - Dice distance

  • ‘jensenshannon’ - Jensen-Shannon distance

  • ‘kulczynski1’ - Kulczynski 1 distance

  • ‘rogerstanimoto’ - Rogers-Tanimoto distance

  • ‘russellrao’ - Russell-Rao distance

  • ‘sokalmichener’ - Sokal-Michener distance

  • ‘sokalsneath’ - Sokal-Sneath distance

  • ‘yule’ - Yule distance

Custom Scipy Metrics:

For advanced usage, use distance='scipy' with additional parameters:

# Minkowski distance with p=3
cluster(data, distance='scipy', metric='minkowski', p=3)

# Mahalanobis distance with custom covariance matrix
cluster(data, distance='scipy', metric='mahalanobis', VI=cov_matrix)

# Any other scipy metric with custom parameters
cluster(data, distance='scipy', metric='your_metric', **kwargs)
